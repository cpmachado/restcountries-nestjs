# restcountries-nest

## Description

restcountries-nest is a reimplementation of
[apilayer/restrountries](https://github.com/apilayer/restcountries), using
[nestjs](https://nestjs.com).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
